### Current Work

### 📊 Exploratory Data Analysis – EM-DAT Disaster Dataset
An in-depth analysis of the EM-DAT disaster dataset focusing on understanding global disaster trends and patterns.

The analysis includes:
- Univariate analysis (distribution and summary statistics)
- Bivariate analysis (relationships and comparisons)
- Multivariate analysis (cross-variable patterns)
- Visual exploration of key trends

The work is documented using reproducible reporting techniques for clarity and structure.
